## 🎯 Rodando a aplicação

Clone o repositório

```bash
git clone https://github.com/MatheusRamalho/burguer-delivery-app.git
```

Utilize o **npm** para instalar as dependências do projeto.

```bash
npm install
```

Em seguida, inicie o projeto.

```bash
npx expo start
```
